## 🎮 Place Your First Bet in 3 Steps

### Step 1: Choose a Match
```
1. Click "Live" or "Upcoming"
2. Browse matches (filter by sport if needed)
3. Click any odds value
```

### Step 2: Set Your Stake
```
1. Bet slip appears on the right
2. Enter stake amount (e.g., $100)
3. See potential winnings calculate automatically
```

### Step 3: Place Bet
```
1. Click "Place Bet"
2. Get unique bet ID (e.g., BET-ABC123-XYZ789)
3. See confirmation with details
```

## 🐛 Troubleshooting

### Matches Not Showing?
- Refresh the page
- Clear browser cache
- Check filter settings

### Bet Won't Place?
- Verify stake amount > 0
- Ensure selections are added
- Try again

### Odds Not Updating?
- Demo shows static odds (expected)
- Live bets stay the same until placed
